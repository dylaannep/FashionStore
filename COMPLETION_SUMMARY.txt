================================================================================
                           KEY FEATURES
================================================================================

1. SHOPPING CART
   - Add items from product pages
   - Update quantities (+/- buttons)
   - Remove individual items
   - Clear entire cart
   - Persistent (localStorage)
   - Real-time calculations (subtotal, IVA, total)

2. CHECKOUT FLOW
   - Step 1: Shipping Information (address, phone, notes)
   - Step 2: Payment Method Selection
   - Step 3: Order Confirmation
   - Form validation at each step
   - Order creation with backend integration

3. ORDER MANAGEMENT (CUSTOMER)
   - View list of personal orders
   - View order details
   - Track order status
   - See shipping information
   - Read-only (cannot modify)

4. ORDER MANAGEMENT (ADMIN)
   - View all orders
   - View complete order details
   - Status transitions:
     * Pendiente → Confirmado
     * Confirmado → Enviado
     * Enviado → Entregado
     * Any state → Cancelado
   - Cancel orders (restores inventory)
   - Track inventory movements

5. INVENTORY MANAGEMENT
   - Automatic stock decrement on order
   - Stock validation before order creation
   - Stock restoration on cancellation
   - Movement logging for all changes
   - Prevents negative stock

                            ARCHITECTURE

DATA FLOW:
  ProductoDetallePage → CartContext → CartPage → CheckoutModal → Backend API

STATE MANAGEMENT:
  - CartContext (React Context): Global cart state

COMPONENTS:

  - CheckoutModal: Multi-step checkout form
  - MisPedidosPage: Customer order list

  - PedidosPage: Admin order list
  - PedidoDetallePage: Admin order management

DATABASE:
  - Pedidos table: Orders
  - DetallePedido table: Order items
  - InventarioProductoVariante table: Stock tracking
  - MovimientoInventario table: Inventory changes
